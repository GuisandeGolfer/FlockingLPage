Flocking Simulator

Welcome to my autonomous agent showcase. I was inspired by Daniel Shiffman's Nature of Code to create a page that has vehicles moving around on the page and tracking your mouse cursor. If you click on the window to the right, you'll be able to see an example of the code that illustrates the concepts that allows for the roaming triangles attacking your cursor.

This animation is heavily based in vectors, which I hope can build my understanding of linear algebra and vector application in machine learning.

I am using p5.JS to overlay this sketch on top of this page. I hope to beable to exercise more creative applicaitons with this library using text, video, webcam, and sound in the browser.
